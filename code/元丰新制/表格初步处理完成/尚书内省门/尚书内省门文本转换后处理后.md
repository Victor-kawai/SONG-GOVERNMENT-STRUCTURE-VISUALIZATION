# 尚书内省门
### 宫官 
宫人女官总称，有别于内命妇及无职事的初选入宫宫人。宋初，宫官名称较杂乱。宋仁宗以后，复遵隋唐旧制，形成了以尚书内省六尚二十四司、二十四典、二十四掌为中心的宫官体制。《旧唐书•职官志》三《宫官》：“六尚，如六尚书之职掌。”《宋会要•后妃》4之2：“宫人女官品：六尚书正五品，二十四司、司正、彤史正七品，二十四掌正八品。”
#### 别称
 女官，即宫人女宫。《宋会要•后妃》4之2：“宫人女官品：六尚书正五品。”《宋诏令》卷22《典簿韩氏司言等制》：“具官某氏等，宫官六尚之属，虽以品秩为差，非恩俾迁焉，亦何由而进乎此？”
### 内省 
管理后宫官人事务的机构名。太祖朝已置（《宋会要•后妃》4之1《内职》）。
### 知内省事 
差遣官名，总掌内省事务，由女官（尚宫及太监）担任。《宋会要•后妃》4之1《内职》：“太宗置尚宫及太监，并知内省事。”
### 尚书内省
#### 沿革 
①内省改名，太宗朝改（《宋会要•后妃》4之1《内职》）。②徽宗朝政和三年五月改定尚书内省职事，确立尚书内省。其下分六司，拟外朝尚书省六部，专以收受尚书省六部所上奏事（《宋诏令》卷21《改定尚书内省职掌御笔政和三年五月二十八日》，及《宋史•徽宗纪》3）。
#### 别称
 ①内尚书省。《玉堂杂记》：“自来宰执兼领书局多是内尚书省批出局名。”《宋会要•职官》2之
## /page{15R}
26《通进司》：“凡内外奏覆必关二司。然后进外，则内官与枢密吏人主掌；内则尚书内省籍其数以下有司。”②大内内侍省。即尚书内省，真宗时见称（《宋诏令》卷24《大内内侍省尚宫赵氏赠崇德夫人制》）。
### 知尚书内省公事 
后宫差遣官名，总掌尚书内省事务，由宫人女官担任。太宗朝时始置。《宋会要•后妃》4之1《内职》：“（太宗）改内省为尚书内省。……司记、知尚书内省公事，皆赐以裙帔。”
#### 简称
 ①知尚书内省事。《宋会要•后妃》4之19：“知尚书内省事、和义郡夫人王从谨。”②知尚书内省。同前书4之13：“主管大内公事、知尚书内省、兼提举十阁分、嘉国懿康惠徽夫人朱氏。”

### 宫人 
宫女。择公卿大夫及良家女入后宫，以备选充御侍或遴选嫔嫱者，通称宫人。从狭义上说，宫人，属未有品位的宫女。从广义上说，也包括有位号之宫女。宫人，未必都是供皇帝使用，皇太后宫、太皇太后宫皆有宫人。宋制规定，年十二、三以下的良家女或公卿大夫之女，经严格挑选、医工审验，方能入宫（《司马光奏议》卷12《后宫等级札子嘉祐八年十二月二日上》）。《宋史•后妃传》上：“冯贤妃，……以良家女，九岁入宫。及长，得侍仁宗。”《宋史•后妃传》上：“李宸妃，杭州人也。……薨，年四十六。初，章献太后欲以宫人礼治丧于外。”按：李宸妃为仁宗生母。章献太后将真相隐瞒，宸妃死时，拟把她当无名号的一般宫人治丧，企图继续瞒住仁宗。
#### 别称
 内人。《宋会要•后妃》4之7：“诏内人张氏特授典赞。”《宋史•钦宗纪》：“（靖康二年）金人以帝及皇后、皇太子北归。……天下州府图书及官吏、内人、内侍、技艺、工匠、娼优、府库蓄积，为之一空。”

宫人的一种，即尚未获取俸禄的宫人，其地位略高于初入选宫人，其上可迁为听宣或红霞帔之类，即可支取请受（俸禄）。《宋会要•后妃》4之29：“未出请受官身杨氏、张氏、史氏并与转听宣。”“未出请受官身田氏并与转红霞帔，并依禄式支破请给。”
