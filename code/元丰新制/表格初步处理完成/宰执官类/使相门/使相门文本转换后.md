# 使相门
### 使相 
加官名。
#### 职源与沿革
 ①出使在外而带丞相之名，始于西汉
初樊哙、尹恢，此为唐五代使相的起源（《玉海》卷
120《汉丞相》、《历代职官表》卷2《内阁》之《历代
建置》）。②使相之名，从唐玄宗朝起，始与宰相分
列。凡带三省长官（尚书令、中书令、纳言或侍中）
及同中书门下平章事或同中书门下三品，在外任
节度使或他官者，都列为使相（《唐会要》卷1）。
③宋初，凡节度使、枢密使、亲王、留守、检校官兼
中书令、侍中、同中书门下平章事，为使相（《长编》
卷17，开宝九年二月庚戌）。④元丰改制，易为开
府仪同三司带节度使为使相（《通考•职官》18
《开府仪同三司》）。
#### 品位
 官品依本官。设使相，主要用于优待勋贤故
老及宰相罢任者。
#### 职掌
 使相不参预政事，凡除授将、相等制敕，在敕尾
不署名，注一“使”字（《宋会要•职官》1之16）。
#### 简称与别名
①相。《宋朝类苑》卷27《使相、节度
使不领京师官局》：“祖宗朝，使相、节度使未尝有
领京师官局者，……至明道中，钱相始为景灵宫
使。”②丞相。《二老堂诗话》卷下《辨欧阳公释奠
诗》：“（钱）惟演止是使相。……亦呼‘丞相’。”③
## /page{89R}
相国。同上书卷：“（钱）惟演止是使相。……是又
呼相国。”④使弼。《宾退录》卷 4：“节度使、检校
太傅而不加辨章（平章），亦非使弼。”⑤宰相。
《二老堂诗话》卷下《辨欧阳公释奠诗》：“此是唐
使相亦谓之宰相。兹乃丞相、相国、宰相三者，在
使相皆可呼之明证。”
### 节度使、同中书门下平章事 
官名。
凡节度使带同中书门下平章事者，非真宰相，为使
相。其制源于唐。北宋元丰改官制，同中书门下
平章事易以开府仪同三司，亦带节度使，称使相
（《石林燕语》卷4）。
#### 别名
 ①节相。《曾巩集》卷23《节相制》。《宋史•
宋库传》：“三曰使相节度，不隶藩要。夫节相之
建，或当边镇，或临师屯。”②衮钺。衮，指相职；
钺，节钺，指节度使。《旧闻证误》卷2：“熙宁五年
三月戊戌，富弼授司空兼侍中，致仕，缺书名。按：
富公实以衮钺挂冠，此云左貂，盖误。“《宋史•富
弼传》：“王安石用事，拜武宁军节度使、同中书门
下平章事。”
